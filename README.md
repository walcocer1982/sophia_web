# Sophia Web

Es una aplicación hecha con NextJS 15.5 (App Router)

## Server UP in dev mode:
- .env
```env
DATABASE_URL="..."

AUTH_SECRET="..."
NEXTAUTH_URL="http://localhost:3001"

GOOGLE_CLIENT_ID="..."
GOOGLE_CLIENT_SECRET="..."
```

- server:
```bash
npm install
npm run dev
```
